SmartDiet - Gestión Nutricional

SmartDiet es una aplicación web diseñada para facilitar la gestión de la alimentación diaria. Integra una base de datos propia con la API externa Open Food Facts, permitiendo consultar información nutricional detallada, registrar consumos y organizar la despensa de forma sencilla. Su enfoque está en promover hábitos alimenticios conscientes mediante una experiencia breve, intuitiva y segura.

Funcionalidades principales

  • 📊 Recomendaciones personalizadas: Cálculo automático en base a perfil, objetivos y registros del usuario.
  • 🥫 Gestión de despensa: Añadir, editar y controlar existencias con datos nutricionales asociados.
  • 🍽️ Registro de consumos: Seguimiento detallado del historial de ingestas diarias.
  • 🔍 Búsqueda de productos: Consulta de información desde base de datos interna y API externa con filtrado y validación de datos.

Tecnologías y arquitectura

  • 🧩 Arquitectura Hexagonal (Ports & Adapters). Separación clara entre dominio, aplicación e infraestructura.
  • 🔐 Seguridad: Autenticación con Spring Security y JWT, garantizando sesiones sin estado y protección de datos.
  • 💾 Persistencia: MySQL con Spring Data JPA.
  • 🌐 Frontend: Thymeleaf + JavaScript (Fetch API) para vistas dinámicas y experiencia fluida.
  • ⚙ Backend: Java 17, Spring Boot 3, API REST segura y modular.

Detalles técnicos

  • Dominio desacoplado de frameworks.
  • Servicios principales: AuthService, DespensaService, ProductoService, RecomendacionNutriService.
  • Capa de seguridad con JwtTokenProvider, JwtAuthenticationFilter y manejo centralizado de errores.
  • Integración de datos locales y externos con almacenamiento en caché para optimizar rendimiento.
Tecnologías: Java, Spring Boot, MySQL, JavaScript, Arquitectura Hexagonal, JWT
